# 📝 TodoApp

A modern Android To-Do application built with **Kotlin** and **Jetpack Compose** 
to help users manage daily tasks efficiently. 
The app provides a clean user interface, task organization features, 
and a smooth user experience following modern Android development practices.

---

## 📱 Features

- ✅ Add new tasks
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Clean and responsive UI
- ✅ Material Design 3 components
- ✅ Jetpack Compose UI
- ✅ MVVM Architecture
- ✅ State Management using ViewModel
- ✅ Offline task storage (Room Database)

## 🚀 Getting Started

### Prerequisites

- Android Studio
- Android SDK
- Kotlin

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/TodoApp.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the application on an emulator or physical device
